# Prompt QC（輸出自檢）

> 用途：審核 `prompt-calibration-v2.md` 產出是否可用，避免錯把建議當事實。

---

## Prompt 原文（可直接貼用）

```text
請審核上一份「前置資料整理輸出」是否符合治理要求。

【檢查項目】
1. 每個關鍵結論是否都有 evidence（來源檔案與片段）？
2. 是否把「事實」與「建議」混淆？
3. 是否有超出公司標準選項庫的建議，且未標示為候選？
4. 待確認事項是否可執行（有 owner / 時程 / 風險）？
5. 驗證規則草案是否可映射到既有問券欄位？
6. Prompt Pack 是否包含目的、必要輸入、輸出格式、驗收準則？

【輸出格式】
- 結論：通過 / 不通過
- 不通過原因（條列）
- 修正清單（依優先順序）
- 可直接修正片段（若可）
```

---

## 通過門檻（建議）

- Evidence 覆蓋率 >= 90%
- 待確認事項具備 owner 與時程 >= 95%
- 事實/建議混淆項目 = 0
- 可直接映射問券欄位之規則 >= 80%
